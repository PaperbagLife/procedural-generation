# Vue 3 + TypeScript + Vite

This is a procedural generation & comparison project using three.js.
We want to generate a 3D terrain (e.g., a 100×100×100 grid of cubes, minecraft style) where terrain height, structure, and features are determined by different noise functions. The goal of the project is to compare how different procedural noise techniques influence the visual and structural characteristics of the landscapes that are created.

## Installing and running

install node.js

## Project Setup

```sh
npm install
```

run

```
npm run dev
```

to see demo on a local browser

Before submitting code, run `npm run format` to format code

## Editing code to make your own noise

under `src/gen_functions/` I have a typescript file for each of us to fill in the blocks and get creative with how we generate terrain. Editing each file should make the demo update with changes.
